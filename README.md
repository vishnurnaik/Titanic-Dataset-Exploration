# Titanic Dataset Exploration

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn.

## Project Overview

The Titanic dataset is one of the most popular datasets used for data analysis and machine learning practice.  
This project focuses on:

- Data cleaning
- Handling missing values
- Statistical analysis
- Data visualization
- Understanding passenger survival patterns

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset

Dataset used:
- Titanic `train.csv`

Dataset contains information such as:
- PassengerId
- Name
- Age
- Gender
- Ticket Fare
- Passenger Class
- Embarked Location
- Survival Status

## Exploratory Data Analysis Performed

- Checking dataset information
- Finding missing values
- Data cleaning
- Survival analysis
- Gender-wise analysis
- Age distribution
- Passenger class analysis
- Correlation analysis
- Visualization using charts and graphs

## Visualizations

The project includes:
- Count plots
- Histograms
- Heatmaps
- Bar charts
- Box plots
